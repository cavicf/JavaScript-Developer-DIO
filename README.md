# Formação JavaScript Developer

Este repositório é dedicado aos estudos, exercícios práticos e projetos desenvolvidos durante a **Formação JavaScript Developer** na plataforma [DIO (Digital Innovation One)](https://www.dio.me/).

---

### 🎯 Objetivo do Repositório

A proposta deste repositório é reunir todo o material produzido ao longo da formação, incluindo códigos-fonte, desafios de código, desafios de projeto e anotações. Ele serve como registro de aprendizado e como portfólio pessoal.

---

### 📚 Conteúdos Abordados

A formação ensina JavaScript do zero, utilizando a técnica de **pair programming** para uma abordagem imersiva e prática. Os principais tópicos estudados incluem:

**Módulo 1 - Aprenda a Programar do Zero com JavaScript:**
- 🚀 Primeiros Passos com JavaScript
- 📝 Variáveis e Tipos de Dados
- 🔄 Estruturas Condicionais e de Repetição
- 🧮 Funções e Escopo
- 📦 Objetos e Arrays

**Módulo 2 - JavaScript para Web: Fundamentos de HTML, CSS e APIs na Prática:**
- 🌐 Introdução ao Desenvolvimento Web com JavaScript
- 🎨 Criando um Projeto com HTML/CSS para Listagem de Pokémon
- 📡 Dominando o Protocolo HTTP e Integrando com a PokeAPI
- 🔴 **Projeto:** Construindo uma Pokédex com JavaScript

**Módulo 3 - Dominando Conceitos Avançados em JavaScript:**
- 📝 Aprendendo Variáveis, Escopo e Tipos de Dados no JavaScript
- 🧬 Orientação a Protótipo com JavaScript
- 🔧 Funções e Manipulação de Listas com JavaScript
- ⚡ Promises e Assincronismo com JavaScript

**Módulo 4 - Projeto Final:**
- 💼 Criação de Currículo Online com a Identidade Visual DIO

---

### 🛠️ Tecnologias Utilizadas

- **Linguagem**: JavaScript
- **Ferramentas**: VS Code, Node.js
- **Conceitos**: DOM, APIs REST, Fetch, Promises, Async/Await, Protótipos

---

### 📁 Organização do Repositório

O repositório está organizado em pastas por módulos:

- 📄 **Módulo 1 - Básico**: Fundamentos do JavaScript e primeiros passos na linguagem.
- 🌐 **Módulo 2 - Web**: JavaScript para Web com HTML, CSS e integração com a PokeAPI.
- 🚀 **Módulo 3 - Avançado**: Conceitos avançados como protótipos, funções e assincronismo.
- 🏆 **Módulo 4 - Projeto Final**: Currículo online com identidade visual da DIO.
